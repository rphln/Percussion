    split(text)

    View Source 
    split(String.t()) :: [String.t()]

    Tries splitting the given text with OptionParser.split/1, and falls back to String.split/1 if it fails.

    Examples 

    iex> Percussion.Utils.split(~s[choose "lorem ipsum" "hello world"])
["choose", "lorem ipsum", "hello world"]

iex> Percussion.Utils.split(~s[I'm a bot])
["I'm", "a", "bot"]
